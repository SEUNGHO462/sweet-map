# 기초 웹 프로그래밍

대학교 기초 웹 프로그래밍 수업에서 학습한 HTML 기반 실습 내용을 정리한 저장소입니다.  
HTML의 기본 구조부터 시맨틱 태그, 입력 양식(Form)까지 단계적으로 학습한 내용을 기록했습니다.

---

## 📌 학습 목표
- HTML 문서의 기본 구조 이해
- 시맨틱 태그를 활용한 의미 있는 웹 문서 작성
- 입력 양식(Form)을 활용한 사용자 입력 처리

---

## 🛠 사용 기술
- HTML5
- Visual Studio Code
- Google Chrome

---

## 📅 주차별 학습 내용 정리

### 9주차 – HTML 문서 구조
- HTML 기본 문서 구조 이해
- 블록 요소(block)와 인라인 요소(inline)의 차이 학습

### 10주차 – id와 class
- id는 문서 내에서 유일하게 사용
- class는 여러 요소에 공통 스타일 적용
- id와 class의 역할 및 사용 목적 구분

### 11주차 – 시맨틱 태그
- header, nav, main, section, article, footer 학습
- 의미 있는 태그를 활용한 문서 구조화
- 시맨틱 태그를 사용한 가독성과 접근성 향상

### 12주차 – 리스트와 표
- ul, ol, dl 태그를 이용한 리스트 작성
- table, tr, th, td를 활용한 표 구성
- rowspan, colspan을 이용한 셀 병합

### 13–14주차 – 입력 양식(Form) 실습
- form 태그를 사용한 사용자 입력 양식 구성
- label 태그를 활용한 입력 요소 설명
- input 태그(type=text, tel, radio) 사용
- textarea를 이용한 여러 줄 텍스트 입력
- placeholder, rows, maxlength 속성 활용

---

## 📝 실습 예제

### 프런트엔드 개발자 지원서 폼
- 이름, 연락처, 지원 분야, 지원 동기를 입력받는 HTML 폼 구현
- 사용자 접근성을 고려하여 모든 입력 요소에 label 태그 적용
- 입력 목적에 따라 input과 textarea를 구분하여 사용

---

## ✍️ 배운 점
- HTML은 단순히 화면을 구성하는 것이 아니라 의미를 표현하는 언어임을 이해했다.
- 시맨틱 태그를 사용하면 문서 구조를 명확하게 표현할 수 있다는 것을 알게 되었다.
- form과 label을 활용하여 사용자 입력을 체계적으로 설계할 수 있게 되었다.
